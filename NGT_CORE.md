# NGT_CORE.md — The Next Geeg Think: Core Identity & Rules

> **This file is sacred.** Any AI agent, developer, or contributor working in this repository MUST read this file first. It contains the synthesized, distilled identity of NGT. Never override, contradict, or ignore the principles here. This is the project's long-term memory.

---

## 1. What Is NGT?

**The Next Geeg Think (NGT)** is a **cognitive infrastructure company** — not a course platform, not a tutorial app, not a traditional edtech business.

"NGT" is a **deliberate misspelling of "the next big thing"** — because the next big thing isn't the product. It's the person using it.

NGT builds **digital systems** that make ambitious people more capable: smarter, more structured, more independent. Every product NGT makes is thinking infrastructure.

---

## 2. The One-Sentence Version

> **NGT builds tools that turn Indonesian students — and eventually the world — into the next big thing, starting with how they think and ending with what they build.**

---

## 3. Mission

> To make every person who touches an NGT product more capable than they were before — not because of NGT, but because of themselves.

NGT does not hand people answers. It hands people better questions, better frameworks, and better systems — then gets out of the way.

---

## 4. Vision

> To become the foundational cognitive infrastructure empowering the next generation of thinkers, creators, and researchers — starting in Indonesia, scaling globally.

---

## 5. Brand Philosophy (The Core Filter)

**"You are the next big thing."** — This is not a tagline. It is a design constraint.

Every feature, every product decision, every line of code must pass this filter:

> *Does this make the user more capable, or does it make them more dependent?*

**If it creates dependency, it does not ship.**

This applies to AI features especially. NGT AI does not write essays. It asks: *"What is your argument?"* NGT AI does not solve problems. It asks: *"What have you tried?"*

---

## 6. Core Values (Non-Negotiable)

| Value | Rule in Practice |
|-------|------------------|
| **Capability Over Convenience** | Ship features that make users better, not tasks easier |
| **Restraint as Discipline** | The AI does less on purpose. The brand speaks less and builds more |
| **The Long Game** | This is a decade-long project. Build for the destination, not a quick win |
| **Student First, System Second** | Every institutional feature exists to serve the student's growth. Never the reverse |
| **Honest About Limits** | Identify the real problem — even when uncomfortable — and build toward solving it |
| **Human Potential Over Automation** | Technology must augment thought, reduce friction, preserve originality, and strengthen agency |

## 7. Product Pipeline

NGT is a **holding company**. Each product has its own brand, audience, and purpose — but they form a deliberate pipeline:

```
THINK BETTER → WRITE BETTER → LEARN BETTER → BUILD BETTER
```

| Product | Status | Audience | Purpose |
|---------|--------|----------|---------|
| **The Nexus** | ✅ Live | High schoolers (SMA) | Notion-based personal OS to organize thinking, tasks, life — building the habit of structured thought |
| **Project Arkais** | 🔨 In Development | College students, academics | AI-augmented research & writing framework — 99% humanized output, preserving the student's intellectual ownership |
| **NGT Spark** | 📐 MVP Planning | Youth, early-career learners | Micro-challenge & portfolio platform — proof-of-potential engine |
| **NGT LMS** | 📅 Concept | All students, schools, institutions | National-scale Learning Management System with AI that interrogates rather than generates |

**Sequencing Rule:** The Nexus is the seed. Arkais is the proof of concept. The LMS is the destination.

---

## 8. Target Audience

**Primary (Indonesia-first):** Youth aged 16–25, university students, early-career professionals, academic researchers.

**Secondary (Long-term):** Global learners and knowledge workers seeking advanced cognitive systems.

These people:
- Seek structured growth, not passive content
- Want intellectual leverage, not shortcuts
- Require systems for clarity and execution

## 9. Strategic Positioning

**NGT is NOT:**

## 11. The Problem NGT Exists to Solve

Two critical failures in education:

1. **Content Without Capability** — Students learn *what* to think, never *how* to think. They memorize but cannot create, critique, or build.
2. **Tools That Do the Work For You** — Most edtech makes tasks easier, not students better. AI accelerates this dependency.

NGT's answer: Products that serve growth, not convenience. Every NGT tool makes the user work — *better*, not *less*.

## 13. Rules for AI Agents & Developers Working in This Repo

1. **Always check this file first.** Any feature, copy, design, or architectural decision must align with NGT's core identity.
2. **Never make NGT sound like a standard edtech SaaS.** It is not. Reject language that commoditizes the brand.
3. **The homepage is a flagship.** It is not a minimum viable product. It reflects the brand's ambition and quality ceiling.
4. **Design must match philosophy.** The homepage should feel as unconventional as NGT itself — bold, unexpected, premium.
5. **This is a long-term project.** Avoid shortcuts that create technical debt. Build to scale.
6. **When in doubt, ask:** *"Does this make the user feel more capable, or more impressed by us?"* — build for the former.
